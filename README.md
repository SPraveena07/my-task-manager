My Day Plan - Task Management App
My Day Plan is a simple yet powerful task management web application designed to help users organize and track their daily activities efficiently.

Features
Task Management: Easily add new tasks, track progress, and delete them when completed.

Priority Levels: Organize tasks based on importance using Low, Medium, or High priority tags.

Voice Commands: Integrated microphone support allows users to add tasks using voice-to-text.

Responsive Design: A modern, clean UI that works perfectly on mobile, tablet, and desktop screens.

Language Toggle: Supports both Tamil and English for a wider range of users.

Tech Stack
HTML5: For the application structure.
CSS3: For custom styling and responsive layouts.
JavaScript (Vanilla): For dynamic task handling, microphone integration, and strict logic validation.
